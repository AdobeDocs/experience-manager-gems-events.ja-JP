# AEM 6.0 開発者モード{#aem-developer-mode}

AEM 6.0 の開発者モードについて David が紹介します。開発者モードは、次のことが可能です。

* 作成者と開発者（コンポーネントツリー）の間の橋渡し
* デバッグヘルパー（エラー通知）
* テストコンパニオン（アプリケーションテストツール）（Hobbes.js AEM 実装）
