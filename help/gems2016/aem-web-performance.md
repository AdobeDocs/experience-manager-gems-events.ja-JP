# AEM Web パフォーマンス{#aem-web-performance}

複雑な Web エクスペリエンスをすばやく実現するのは困難です。 この講演では、ジョエルはより一般的な落とし穴のいくつかに光を当てます。 理論、例、ベストプラクティスに加えて、ネットワークとレイテンシに関するツールのデモが含まれます。HTTP/1.1 および HTTP/2 の最適化 — AEM Server Requests の処理が遅くなる問題の識別 — JavaScript:オーバーヘッドの最小化
